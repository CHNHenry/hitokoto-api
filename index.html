🎯 Hitokoto API

基于 GitHub Pages 的静态一言 API

API 端点

GET https://你的用户名.github.io/hitokoto-api/hitokoto.json

使用示例

// 前端随机获取
fetch('https://你的用户名.github.io/hitokoto-api/hitokoto.json')
  .then(r => r.json())
  .then(data => {
    const random = data[Math.floor(Math.random() * data.length)];
    console.log(random.content);
  });
